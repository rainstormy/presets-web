= CSS Preset for Prettier
:experimental:
:source-highlighter: highlight.js

This package provides a predefined, opinionated https://prettier.io[Prettier] configuration suitable for any https://www.typescriptlang.org[TypeScript]-based web project with CSS declarations.

This preset includes the following plugin, which is installed along with the preset package:

* https://github.com/Siilwyn/prettier-plugin-css-order[prettier-plugin-css-order]

== Installation
Install the `@rainstormy/preset-prettier-css` package and its peer dependencies:

[source,shell]
----
npm install --save-dev @rainstormy/preset-prettier-css postcss prettier
----

[source,shell]
----
pnpm install --save-dev @rainstormy/preset-prettier-css postcss prettier
----

[source,shell]
----
yarn add --dev @rainstormy/preset-prettier-css postcss prettier
----

== Usage
Create or extend a https://prettier.io/docs/en/configuration.html[Prettier configuration file] (`prettier.config.js`) and apply the `prettierPresetCss` preset.

For example:

[source,javascript]
----
import { prettierPresets } from "@rainstormy/preset-prettier-base"
import { prettierPresetCss } from "@rainstormy/preset-prettier-css"

export default prettierPresets({
	additionalPresets: [
		// ...other presets...
		prettierPresetCss(),
	],
})
----

=== In Combination with Tailwind CSS
For example:

[source,javascript]
----
import { prettierPresets } from "@rainstormy/preset-prettier-base"
import { prettierPresetCss } from "@rainstormy/preset-prettier-css"
import { prettierPresetTailwind } from "@rainstormy/preset-prettier-tailwind"

export default prettierPresets({
	additionalPresets: [
		prettierPresetCss(),
		prettierPresetTailwind(),
	],
})
----
